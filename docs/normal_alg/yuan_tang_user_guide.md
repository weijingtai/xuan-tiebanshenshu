## 1. 概述

### 1.1 什么是元堂卦取数法？

元堂卦取数法是铁板神数中的一种重要算法，通过**四柱八字**、**性别**、**三元**、**节气**等信息，推算出**先天卦**和**后天卦**，并基于此计算一生的**大运**、**流年**和**流月**运势。

### 1.2 核心特点

- **三层运势系统**：大运（6-9年/期）→ 流年（1年/卦）→ 流月（1月/卦）
- **动态卦象变化**：通过爻变展示运势转换过程
- **精准时间定位**：可查询任意年龄、任意月份的具体运势
- **可视化展示**：六爻图形化展示，直观易懂

## 2. 使用前准备

### 2.1 必需信息

1. **出生年月日时**（农历）
   - 需要准确到时辰
   - 系统会自动转换为四柱八字

2. **性别**
   - 男 / 女
   - 影响卦象的上下排列

3. **三元**
   - 上元 / 中元 / 下元
   - 用于天数或地数为5时的特殊处理

4. **出生节气**
   - 夏至后 / 冬至后
   - 影响六阴爻或六阳爻的装卦方式

### 2.2 信息获取方式

- 使用App内置的"万年历"功能查询农历生日
- 通过"节气查询"确定出生在夏至后还是冬至后
- 三元信息可咨询专业术数老师

## 3. 基础计算结果

### 3.1 先天卦与后天卦

计算完成后，首先会看到两个核心卦象：

**先天卦概览**
```
先天卦：震坤
  上卦：震(8)
  下卦：坤(5)
  元堂爻：三爻
```

**后天卦概览**
```
后天卦：坎巽
  上卦：坎(6)
  下卦：巽(7)
  元堂爻：四爻
```

**解读说明**：
- 先天卦：代表先天命格，出生时的能量状态
- 后天卦：代表后天运势，人生的发展轨迹
- 元堂爻：最关键的爻位，是运势变化的核心

### 3.2 卦象可视化

每个卦象都会显示六爻结构（从上到下）：
```
上爻 ━━━━━  (阳爻 - 实线)
五爻 ━  ━  (阴爻 - 断线)
四爻 ━━━━━
三爻 ━  ━  ★元堂 (标记元堂爻)
二爻 ━━━━━
初爻 ━  ━
```

- **阳爻**：实线（━━━━━），代表阳刚、主动
- **阴爻**：断线（━  ━），代表阴柔、被动
- **元堂爻**：标有星标，是该卦最重要的爻位

## 4. 流运系统详解

### 4.1 大运系统

**什么是大运？**
- 人生按照六爻划分为12个大运期
- 先天卦6个大运期 + 后天卦6个大运期
- 每个大运期持续时间：
  - 阳爻大运：9年
  - 阴爻大运：6年

**大运表格示例**：
```
先天卦大运（震坤）
┌────┬────┬─────┬───────┬─────────┐
│爻位│阴阳│年数 │年龄区间│  地支   │
├────┼────┼─────┼───────┼─────────┤
│三爻│阴  │6年  │1-6岁  │辰、巳   │
│四爻│阳  │9年  │7-15岁 │午、未、申│
│五爻│阴  │6年  │16-21岁│酉、戌   │
│...│...│...  │...    │...      │
└────┴────┴─────┴───────┴─────────┘
```

**如何使用**：
1. 找到当前年龄所在的大运期
2. 查看该大运的阴阳属性
3. 了解该大运期的时间跨度
4. 查看配置的地支（与流月卦相关）

### 4.2 流年卦系统

**什么是流年卦？**
- 每个大运期内，每一年都有对应的流年卦
- 阳爻大运：9个流年卦
- 阴爻大运：6个流年卦
- 流年卦通过爻变从基础卦（先天卦或后天卦）推演而来

**流年卡片展示**：
```
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│   7岁 ★    │  │    8岁      │  │    9岁      │
│   震坤      │  │   震艮      │  │   震兑      │
│  变四爻     │  │  变五爻     │  │  变上爻     │
│   (点击)    │  │   (点击)    │  │   (点击)    │
└─────────────┘  └─────────────┘  └─────────────┘
```

**流年卦特点**：
- **大运首年**（如7岁）有星标，是该大运期的开始
- **变爻标注**：显示该年变化的爻位
- **可点击**：点击任意流年卡片，查看该年的12个流月卦

**爻变规则**：
- **阳爻大运**：
  - 初年为阳年：第1年不变换
  - 初年为阴年：第1年变换大运爻
  - 第2-9年：按 (爻-2) → 爻 → (爻+1) → (爻+2) 循环变换

- **阴爻大运**：
  - 不论初年阴阳，第1年都变换大运爻
  - 第2-6年：逐爻变换 (爻+1) → (爻+2) → ... → (爻+5)

### 4.3 流月卦系统

**什么是流月卦？**
- 每个流年内，12个月各有一个流月卦
- 通过点击流年卡片展开查看
- 流月卦区分阳月和阴月

**流月网格展示**：
```
┌─────┬─────┬─────┬─────┐
│1月  │2月  │3月  │4月  │
│阳月 │阴月 │阳月 │阴月 │
│震坤 │震艮 │震兑 │震乾 │
│变二 │变五 │变三 │变上 │
├─────┼─────┼─────┼─────┤
│5月  │6月  │7月  │8月  │
│...  │...  │...  │...  │
└─────┴─────┴─────┴─────┘
```

**流月特点**：
- **阳月**（1,3,5,7,9,11月）：橙色标识，逐月变换下一爻
- **阴月**（2,4,6,8,10,12月）：靛蓝标识，取对应阳月的应爻变换
- **应爻对应**：初↔四、二↔五、三↔上

**流月起法**：
1. **正月卦**：变换（元堂爻-1）的爻位
2. **其他阳月**：从正月卦开始，逐月变换下一爻
3. **阴月卦**：取对应阳月卦，变换其应爻

## 5. 实际操作步骤

### 5.1 查看基础卦象

1. 进入元堂卦结果页面
2. 查看**先天卦和后天卦概览**区域
3. 点击"计算步骤详情"了解详细推算过程
4. 查看六爻地支配置，找到元堂爻位置

### 5.2 查看大运计算

1. 展开"**大运计算（先天卦+后天卦）**"区域
2. 查看先天卦大运表格，找到当前年龄所在大运期
3. 查看后天卦大运表格，了解后续运势分布

### 5.3 查看流年卦

1. 展开"**流运系统**"区域
2. 先天卦流运区域显示前半生的流年卦
3. 后天卦流运区域显示后半生的流年卦
4. 每个大运期的流年卡片横向滚动查看

### 5.4 查看流月卦

1. 在流年卡片列表中，点击想要查看的年龄卡片
2. 自动展开该年的12个流月卦网格
3. 查看每月的卦象和变爻信息
4. 点击"关闭"按钮收起流月详情

## 6. 高级解读技巧

### 6.1 判断大运吉凶

**阳爻大运**：
- 能量强劲，适合主动出击
- 9年周期较长，运势相对稳定
- 关注首年是否变换（阳年起 vs 阴年起）

**阴爻大运**：
- 能量柔和，适合积累沉淀
- 6年周期较短，变化较快
- 首年必定变换，需注意适应

### 6.2 解读流年变化

**关注变爻位置**：
- **初爻、二爻变**：影响家庭、根基
- **三爻、四爻变**：影响事业、人际
- **五爻、上爻变**：影响决策、远景

**观察卦象转换**：
- 从实线（阳爻）变为断线（阴爻）：由刚转柔，宜守不宜攻
- 从断线（阴爻）变为实线（阳爻）：由柔转刚，可适当进取

### 6.3 流月择时

**阳月与阴月**：
- **阳月**（奇数月）：能量外放，适合行动、拓展
- **阴月**（偶数月）：能量内敛，适合思考、整合

**应爻对应原理**：
- 阴月卦取自阳月卦的应爻变换
- 体现了"一阴一阳之谓道"的自然规律

## 7. 条文系统

### 7.1 条文编号方法

元堂卦提供8种条文编号方法：

1. **先天卦加则法**
2. **后天卦加则法**
3. **先天卦纳甲太玄数法**
4. **后天卦纳甲太玄数法**
5. **先天卦本互法**
6. **后天卦本互法**
7. **先天卦互取数列表**（减2、4、8、16倍 + 加则扩展）
8. **后天卦互取数列表**（减2、4、8、16倍 + 加则扩展）

### 7.2 条文扩展

每个基础数递加96四次，得到5个条文编号：
```
基础数: 123
扩展: 123, 219, 315, 411, 507
```

### 7.3 查看条文内容

1. 展开"**条文内容列表**"区域
2. 每条条文显示：
   - 条文编号（如123）
   - 地支标签（如子丑寅卯）
   - 来源标签（如"先天卦扩展1"）
   - 条文正文内容

## 8. 常见问题

### Q1: 为什么我的流年卡片数量不一样？
**A**: 这是正常的。阳爻大运有9个流年卡片，阴爻大运有6个流年卡片。

### Q2: 流月卦的"应爻"是什么意思？
**A**: 应爻是传统六爻术数中的概念，表示对应关系：
- 初爻 ↔ 四爻
- 二爻 ↔ 五爻
- 三爻 ↔ 上爻

阴月卦通过变换对应阳月卦的应爻得到。

### Q3: 为什么有些流年卡片标"未变换"？
**A**: 这出现在阳爻大运且初年为阳年的情况下。第1年直接使用基础卦，不进行爻变。

### Q4: 点击流年卡片没有反应？
**A**: 请确保：
1. 网络连接正常（首次计算需要联网）
2. App已更新到最新版本
3. 系统有足够的内存空间

### Q5: 流月卦计算很慢怎么办？
**A**: 流月卦是按需计算的：
1. 首次点击某个流年会有短暂计算过程（<1秒）
2. 计算后会缓存结果，再次查看即刻显示
3. 如仍然很慢，请尝试重启App

### Q6: 如何判断至尊卦？
**A**: 坎坎、坎震、坎艮三个卦是至尊卦，在元堂爻为九五或上六时，后天卦的生成规则会根据出生月份阴阳有特殊处理。

## 9. 术语解释

- **四柱八字**：年、月、日、时的天干地支组合
- **先天卦**：基于出生信息计算的初始卦象
- **后天卦**：先天卦元堂爻变换后得到的卦象
- **元堂爻**：根据时辰配上地支最多的爻位
- **爻变**：将阳爻变为阴爻，或阴爻变为阳爻
- **大运**：以六爻为周期的运势阶段
- **流年**：某一年的运势卦象
- **流月**：某一月的运势卦象
- **阳爻**：实线爻（━━━━━）
- **阴爻**：断线爻（━  ━）
- **互卦**：取二、三、四爻为下卦，三、四、五爻为上卦组成的新卦

## 10. 使用建议

### 10.1 学习顺序

1. **第一步**：理解先天卦和后天卦的含义
2. **第二步**：熟悉大运系统，知道自己当前处于哪个大运期
3. **第三步**：查看当前流年卦，了解本年运势
4. **第四步**：展开流月卦，择时而动

### 10.2 实际应用

- **年度规划**：每年初查看该年流年卦，制定年度目标
- **月度调整**：每月初查看该月流月卦，调整行动策略
- **重大决策**：查看当前流月卦，选择适宜的时机
- **回顾总结**：查看过往流年流月，验证运势准确度

### 10.3 注意事项

1. **不要过度依赖**：流运系统提供参考，最终决策仍需结合实际
2. **动态调整**：运势会随时间变化，定期查看更新
3. **综合判断**：结合条文内容、卦象变化、个人情况综合分析
4. **保持平和**：好运时不骄，逆境时不馁
