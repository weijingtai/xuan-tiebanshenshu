# 任务队列

> AI MUST NOT 修改此文件，除非用户明确命令 + 许可（详见 docs/ai/board-protocol.md W1-W5 通道）。
> AI MAY 在用户明确命令下读取此文件以同步进度。

## 状态定义

| 标记 | 状态 | 说明 |
|------|------|------|
| `[ ]` | 待认领 | 等待发布人认领并开始实现 |
| `[~]` | 进行中 | 已认领，正在实现 |
| `[!]` | 阻塞 | 被外部依赖卡住，暂停 |
| `[x]` | 已完成 | 验收通过，代码已合并 |
| `[R]` | 已撤销 | 不再需要 |

## 条目格式

```
## TASK-NNN

- 状态: [ ] | [~] | [!] | [x] | [R]
- 类型: feature | refactor | fix | chore
- 标题: <中文简述>
- 发布人: <developer>-<Model>
- 认领人: <developer>-<Model>
- 创建: YYYY-MM-DD
- 完成: YYYY-MM-DD (完成后填写)
- 工作目录: docs/<developer>-<Model>/<type>/<name>-<date>/
- 阻塞原因: (仅状态 [!] 时填写)
- 关联: (关联的其他 TASK-ID，如存在)
```
