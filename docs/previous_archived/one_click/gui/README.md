# One-Click Divination UI Templates

This directory contains the HTML/CSS templates for the Unified Divination System UI.
These templates serve as the **Acceptance Criteria** for the Flutter implementation.

## Structure

* `components/`: Reusable components (Cards, Buttons, Strip Items).
* `pages/`: Full page mockups (UnifiedDivinationPage, Mobile/Desktop layouts).
* `assets/`: Images and icons used in the mockups.

## Delivery Standard

1. **Responsiveness**: Templates must demonstrate how layouts adapt to different screen widths.
2. **Interactivity**: Hover states, active states, and basic transitions should be defined in CSS.
3. **Theming**: Should reflect the "New Chinese Style" defined in the Design Guidelines.
